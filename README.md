# love-peace-backstage



## 技術介紹


前端
* JavaScript

  使用 ES6 JavaScript 語法做開發
  
* Sass/SCSS

  使用Sass開發
  
* React Router

  使用 React Router 實踐 SPA 的 router 管理
  
* React-Bootstrap/Bootstrap

  使用 React-Bootstrap 中的 Component 進行開發
  
<br />

後端
* Node.js、Express.js、Axios

  使用 Node.js、Express.js、Axios 技術撰寫 API 使前端與 API req/res
  
* RESTful API

  實踐 RESTful API 並更新資料庫
  
* MySQL

  使用MySQL建立關聯式資料庫

<br />

開發

* Git/GitHub

  使用 Git/GitHub 做版本控制

* NPM

  使用 NPM、ES6 module system、creat-react-app 導入套件、將前端程式碼模組化

<br />

## 專案介紹

「Love & Peace Rock Festival - 後台系統」提供「Love & Peace Rock Festival」後台資料的顯示、查詢和管理。包含票卷訂單查詢、商品訂單查詢、商品販售報表分析與商品庫存管理，四個顯示頁面。

<br />

* 登入頁 <br />
管理員的登入頁，輸入正確的帳號密碼才能進入後台系統。

<img src="https://github.com/zechengze/love-peace-backstage/blob/main/screenshot/login.png" width="720">

* 票券訂單查詢頁 <br />
顯示網站前台所有會員的票券訂單，上方有搜尋欄提供使用者以類別篩選需要查詢的條件，選取類別，然後輸入條件，就可以獲得相對應的資料列。

* 商品訂單查詢頁 <br />
顯示網站前台所有會員的商品訂單明細，上方有搜尋欄提供使用者以類別篩選需要查詢的條件，選取類別，然後輸入條件，就可以獲得相對應的資料列。點選訂單編號旁的檢視按鈕可以查看個別的商品訂單明細。

* 商品販售報表分析 <br />
  以圖表去顯示過去一年週邊商品在各月份的收益，折線圖代表總收益，長條圖代表淨利。
  
<img src="https://github.com/zechengze/love-peace-backstage/blob/main/screenshot/chart.png" width="720">

* 商品庫存管理 <br />
  顯示所有網站前台販售的週邊商品資料，上方有商品搜尋列可以搜尋商品名稱，點選右上角上架按鈕後會跳出視窗，輸入詳細資訊就可以將產品新增到畫面和資料庫中，資料列右邊有編輯按鈕，點選後可以編輯商品的詳細資訊，點選垃圾桶圖示就可以下架選中的產品。
